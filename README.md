  ## installation

```console

# clone the repo
$ git clone https://github.com/ohhhio/bartolomeuIA.git

# change the working directory to bartholomeusAI
$ cd bartholomeusAI

# install python3 and python3-pip if they are not installed

# install the requirements 
$ pip install ChatterBot

```
